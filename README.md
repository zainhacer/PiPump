## What is PiPump?

PiPump is a **meme token launchpad on Pi Network** — similar to pump.fun on Solana. Anyone can:

- 🪙 **Create tokens** with a 1π fee
- 📈 **Buy & Sell** via automatic bonding curve (AMM)
- 💼 **Track portfolio** and trade history
- 🎓 **Graduate tokens** at 800π collected

## Tech Stack

- **Frontend**: React 18 + Vite + Tailwind CSS
- **Database**: Supabase (Postgres + Realtime)
- **Auth + Payments**: Pi SDK (real Pi wallet)
- **Hosting**: GitHub Pages

## Setup

See **[GUIDE.md](./GUIDE.md)** for complete installation and deployment instructions.

```bash
npm install --legacy-peer-deps
cp .env.example .env
# Fill in .env with your keys
npm run dev
```

## Deploy

```bash
npm run deploy
```
